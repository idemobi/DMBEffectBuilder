# DMBEffectBuilder Glossary

## Purpose

Define common terms used in `DMBEffectBuilder` documentation and AI instructions.

## Project-specific section

When copying this file to another PageBuilder ecosystem project, update this section first.

- Project name: `DMBEffectBuilder`
- Project folder: `DMBEffectBuilder`
- Project role: reusable visual effect package for PageBuilder and BootstrapBuilder applications.
- Publication host: `labs_idemobi_com`

## Terms

### Effect extension

A fluent extension method that applies a visual effect to an existing BootstrapBuilder or PageBuilder object.

### Image effect

An effect applied to image output, such as blur, color shift, glow, reveal, flip, rotate, shake, vignette, or zoom.

### Section effect

An effect or helper applied to a larger page section or used to debug section-level visual behavior.

### Effect class

A generated CSS class that activates one visual effect in the consuming page.

### Reduced motion

An accessibility strategy for limiting movement-heavy effects when a user or consuming site prefers less animation.

### Hover fallback

The non-hover behavior used for touch, keyboard, or devices where hover effects are not available.

### Embedded asset

A static asset included in the package and exposed to consuming MVC/Razor applications through effect configuration.

### DocumentationViewer

The documentation browsing feature in `labs_idemobi_com` that displays generated API documentation for NuGet packages.

### DocumentationBuilder

The documentation generation process that extracts and renders API documentation. AI prepares content; the developer executes the generator.
